**Project Title:** Hoax Detection AI System - Artificial Intelligence UAS Group 18
**Description:** 

This Flask-based web application implements a **Hybrid Hoax Detection System** combining rule-based lexicon matching with statistical probability scoring for real-time Indonesian hoax news classification. The system analyzes news headlines and social media content to classify text as "HOAX" or "VALID" with confidence scores (60-98%).

- 20+ Indonesian hoax keyword lexicon (medical scams, policy hoaxes, natural disaster rumors)
- Real-time web interface with glassmorphism UI design 
- Confidence scoring with hoax bias optimization (threshold 0.42)
- Deployed live on Replit for newsroom monitoring and fact-checking
